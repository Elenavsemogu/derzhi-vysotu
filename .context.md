# Держи Высоту — сайт скалодрома

## Описание
Сайт для скалодрома «Держи Высоту» в Барнауле (ИП Катунин). Полный редизайн существующего сайта (держивысоту.рф) с нуля: разделение аудиторий (дети/взрослые), проработанный копирайтинг, предложения по логотипу.

## Стек
- HTML/CSS (single-page, статика)
- SVG логотипы
- GitHub Pages

## Деплой
- **GitHub:** `Elenavsemogu/derzhi-vysotu`
- **Сайт:** https://elenavsemogu.github.io/derzhi-vysotu/
- **Логотипы:** https://elenavsemogu.github.io/derzhi-vysotu/logo-proposals.html

## Файлы
- `index.html` — основной сайт (split-hero, дети 4-14, взрослые 16+, цены, отзывы, FAQ, CTA)
- `logo-v3.html` — СТРАНИЦА ДЛЯ КСЮШИ: только 2 варианта (без описаний), заголовок-шутка. Это шлём клиенту.
- `logo-elena.html` — рабочая страница для Лены: ещё 3 варианта в том же стиле.
- `logo-proposals.html` — старые 5 концептов (A–E, горы+человечки) — отклонены
- `logo-original.png` — текущий логотип клиента (жест «коза» на фоне гор)
- `diploma.png` — сертификат (референс лаймового цвета)

## Логотип — направление v3
- **Стиль:** чистый геометрический леттеринг «Держи Высоту» со спрятанным скалодромным образом в одной букве (референс — «Выдрочка» с выдрой в «О», THE CLIMBER, «Рыба Люба»). Без мультяшных гор и стик-человечков (их отклонили в v2).
- **Палитра:** салатовый `#A3E635` / лайм `#84CC16` + графит `#26292E`.
- **Шрифт-ориентир:** Unbounded / Montserrat ExtraBold (финал — в вектор).
- **Для Ксюши (logo-v3.html):** Вариант 1 — Карабин в «О», Вариант 2 — Зацеп в «О».
- **Для Лены (logo-elena.html):** Вариант 3 — Мешочек магнезии в «О», Вариант 4 — «Связка» (два кольца) в «О», Вариант 5 — Слово между зацепами. (Старые гора-«Д»/монограмма/стрелка — отклонены Ксюшей.)
